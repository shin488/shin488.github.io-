スライドパズルの歴史

**1874年:** ノイス・チャップマンが考案

**1878年:** サムロイドが「15パズル」として発表し世界的に流行

**20世紀初頭:** 「Klotski」「華容道」「箱入り娘」などのバリエーションが登場

スライドパズルの作り方（HTML + CSS + JavaScript）

📌 **基本設計:** HTMLでグリッドを作成、CSSでデザイン調整

📌 **ピースの配置:** JavaScriptで動的に制御

📌 **スライド機能:** 隣のピースと入れ替えるロジックを追加

📌 **クリア判定:** 正しく並んだら「クリア！」と表示

コード例（HTML + CSS + JavaScript）

📌 `index.html` に以下のコードを保存すれば、スライドパズルが完成！

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>スライドパズル</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>スライドパズル</h1>
    <div id="puzzle"></div>
    <button onclick="shuffle()">シャッフル</button>
    <script src="script.js"></script>
</body>
</html>
